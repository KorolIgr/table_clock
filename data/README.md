# Веб-интерфейс Table Clock

## Структура проекта

Веб-интерфейс состоит из следующих страниц:

- `index.html` - главная страница настройки WiFi
- `time.html` - страница настройки отображения времени
- `leds.html` - страница управления светодиодами
- `settings.html` - страница общих настроек
- `template.html` - общий шаблон (не используется напрямую)

## Система сборки

HTML-файлы автоматически преобразуются в строковые константы C++ с помощью скрипта `build_html.py`. При запуске скрипта:

1. Все HTML-файлы из директории `data/html` преобразуются в строковые константы
2. Константы сохраняются в файле `include/web_pages.h`
3. Каждый HTML-файл становится доступным как константа с суффиксом `_HTML`

Например:
- `index.html` → `INDEX_HTML`
- `time.html` → `TIME_HTML`
- `leds.html` → `LEDS_HTML`
- `settings.html` → `SETTINGS_HTML`

## Использование

Для обновления веб-страниц:

1. Измените нужный HTML-файл в директории `data/html`
2. Запустите скрипт: `python build_html.py`
3. Пересоберите проект

## API эндпоинты

Веб-интерфейс поддерживает следующие эндпоинты:

### Основные
- `GET /` - главная страница (WiFi настройки)
- `GET /time` - страница настройки времени
- `GET /leds` - страница управления светодиодами
- `GET /settings` - страница общих настроек
- `GET /status` - статус WiFi соединения (JSON)

### Настройки времени
- `POST /save_time_settings` - сохранить настройки времени
- `POST /sync_time` - синхронизировать время
- `GET /current_time` - получить текущее время

### Управление светодиодами
- `POST /save_led_settings` - сохранить настройки светодиодов
- `POST /test_leds` - запустить тест светодиодов
- `POST /turn_off_leds` - выключить все светодиоды

### Общие настройки
- `POST /save_general_settings` - сохранить общие настройки
- `GET /device_info` - информация об устройстве
- `POST /reboot` - перезагрузка устройства
- `POST /factory_reset` - сброс к заводским настройкам

### WiFi
- `GET /scan` - сканирование WiFi сетей
- `POST /connect` - подключение к WiFi
- `POST /forget` - забыть сохраненные учетные данные
