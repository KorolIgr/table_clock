* The device must support hybrid WiFi mode (AP + STA)
* For AP, connection data is stored in EEPROM
* If there is no data for the AP in the EEPROM, then the default SSID is the project name and the password is 12345678
* For STA, connection data is stored in EEPROM
* If there is no data for STA in the EEPROM, do not connect to the access point
