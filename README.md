# Table Clock Project

## Описание проекта

Это проект настольных часов на основе контроллера ESP8266 с 8 дисплеями SSD1306 (через мультиплексор TCA9548A) и 8 светодиодами APA106. Устройство подключается к WiFi, имеет веб-интерфейс управления и режим точки доступа для первоначальной настройки.

## Особенности

- Отображение времени на 8 дисплеях SSD1306 128x64
- Управление яркостью и эффектами 8 светодиодов APA106
- Подключение к WiFi сети
- Веб-интерфейс для настройки
- Режим точки доступа для первоначальной настройки
- Возможность синхронизации времени по NTP
- Поддержка OTA обновлений
- Энергоэффективность

## Компоненты

- **ESP8266** (NodeMCU или Wemos D1 Mini) - основной контроллер
- **TCA9548A** - I2C мультиплексор для подключения 8 дисплеев
- **8 дисплеев SSD1306** 128x64 - для отображения информации
- **8 светодиодов APA106** - для световых эффектов
- **Источник питания 5V** - для питания всей системы

## Архитектура

```
ESP8266 (NodeMCU/Wemos D1 Mini)
├── VCC (3.3V) → TCA9548A VCC
├── GND → Все компоненты GND
├── SDA (D2, GPIO4) → TCA9548A SDA
├── SCL (D1, GPIO5) → TCA9548A SCL
├── DIN (D8, GPIO15) → APA106 LED Strip Data Input
└── 5V Power → APA106 VCC (отдельный источник)

TCA9548A (I2C Multiplexer)
├── VCC → 3.3V
├── GND → Ground
├── SDA → ESP8266 D2 (GPIO4)
├── SCL → ESP8266 D1 (GPIO5)
├── Address → GND (I2C address: 0x70)
├── Channel 0 → SSD1306 Display 1 (0x3C) SDA/SCL
├── Channel 1 → SSD1306 Display 2 (0x3C) SDA/SCL
├── Channel 2 → SSD1306 Display 3 (0x3C) SDA/SCL
├── Channel 3 → SSD1306 Display 4 (0x3C) SDA/SCL
├── Channel 4 → SSD1306 Display 5 (0x3C) SDA/SCL
├── Channel 5 → SSD1306 Display 6 (0x3C) SDA/SCL
├── Channel 6 → SSD1306 Display 7 (0x3C) SDA/SCL
└── Channel 7 → SSD1306 Display 8 (0x3C) SDA/SCL

APA106 LED Strip Connection
├── VCC → 5V Power Supply
├── GND → Ground
└── DIN → ESP8266 D8 (GPIO15)
    ├── LED 1 → LED 2 → LED 3 → LED 4 → LED 5 → LED 6 → LED 7 → LED 8
    └── DOUT of each LED connects to DIN of the next LED in sequence
```

## Технические требования

- Поддержка OTA обновлений
- Сохранение настроек в энергонезависимой памяти
- Обработка ошибок подключения
- Энергоэффективность

## Потребляемая мощность

- ESP8266: до 240 мА при WiFi передаче
- Один SSD1306: до 20 мА
- 8 SSD1306: до 160 мА
- Один APA106: до 60 мА
- 8 APA106: до 480 мА
- Общий ток: до 880 мА при полной яркости

## Возможные проблемы и решения

1. Проблемы с питанием - использовать отдельный стабилизированный блок питания 5V
2. Проблемы с I2C шиной - использовать короткие провода и подтягивающие резисторы
3. Проблемы с памятью ESP8266 - оптимизировать использование памяти и использовать PROGMEM
4. Проблемы с синхронизацией - использовать таймеры и прерывания для синхронной работы
